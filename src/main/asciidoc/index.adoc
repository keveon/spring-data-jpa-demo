      <div>
        <span class="section-kicker">Quick start</span>
        <h2 class="section-title">3 分钟完成本地验证</h2>
      </div>
    </div>
    <p class="section-intro">当前维护基线为 <strong>Spring Boot 3.5.13</strong> 与 <strong>Java 21</strong>。</p>
    <div class="quickstart panel-card">
      <p>下面这组命令足够完成环境准备、测试和本地启动：</p>
      <div class="terminal-block">
        <code>mise install
./mvnw test
./mvnw spring-boot:run</code>
      </div>
      <ol>
        <li>执行 <code>mise install</code>，准备项目要求的 Java 21 环境。</li>
        <li>执行 <code>./mvnw test</code>，生成 REST Docs snippets 与测试结果。</li>
        <li>执行 <code>./mvnw spring-boot:run</code>，本地体验接口和文档页面。</li>
      </ol>
    </div>
