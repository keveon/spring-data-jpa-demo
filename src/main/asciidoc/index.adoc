    <div class="quickstart hero-panel">
      <p>推荐使用 <a href="https://mise.jdx.dev/">mise</a> 管理 Java 版本。下面这组命令足够完成安装、测试和本地启动：</p>
      <div class="terminal-block">
        <code>mise install
./mvnw test
./mvnw spring-boot:run</code>
      </div>
      <ol>
        <li>访问仓库根目录后执行 <code>mise install</code>，即可拉起项目要求的 Java 21 环境。</li>
        <li>运行 <code>./mvnw test</code>，会同时生成 REST Docs 片段与覆盖率报告。</li>
        <li>运行 <code>./mvnw spring-boot:run</code>，即可在本地体验接口与页面文档。</li>
